Fonoa hackathon

Challenge: combat information overload from high volumes of long-form information articles.
User needs: Stay informed quickly without spending hours filtering and reading.
Solution: Develop an AI-powered system to automatically collect, filter and summarise news.
Goal: deliver fast, user-centric and trustworthy news presentation

Problem statement

Long articles - prefer summarised version to decide is it worthwhile
Falsely generated information - Require high-confident, reliable content 
Large amount of news - wants to follow only categories of interest 
Manual website checking - receive scheduled updates
Plain text articles - wants richer presentation formats

Constrains 

Use “The Hacker News” website as the data source.

Submission 

1. Team name
2. Solution description (max. 300 characters)
3. Technical solution explanation
4. Solution completeness overview 
5. 1-min demo video URL
6. PowerPoints slides for final pitch
7. Link to Github repository


Scoring Criteria

1. Creativity and Innovation
2. Technical Execution 
3. Business impact 
4. UI/UX Design
5. Demo (presentation)
